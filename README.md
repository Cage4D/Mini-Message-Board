# Mini Message Board

A simple message board application built with Node.js and Express. Users can view existing messages and submit new ones through a form.

## Features

- View a list of messages
- Submit new messages through a form
- Server-side rendering with Pug
- Express routing and middleware
- Simple in-memory data storage

## Built With

- Node.js
- Express
- EJS (template engine)
- HTML & CSS
